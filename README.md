# E-commerce Shopping Cart System (Система електронної комерції)

Програмний комплекс призначений для автоматизації процесів обліку товарів, керування кошиком покупця та безпечного оформлення замовлень під час транзакцій. Проєкт розроблено в межах курсової роботи з дисципліни "Програмування, алгоритмізація та основи промпт-инжинірингу".

## 🚀 Основний функціонал

* **Поліморфна ієрархія товарів**: Розрахунок кінцевої вартості залежно від категорії товару (базовий `Product`, `DigitalProduct` зі знижкою та `PhysicalProduct` з урахуванням вартості доставки).
* **Контроль залишків**: Валідація кількості доступних товарів на складі під час додавання до кошика.
* **Контекстне керування замовленнями**: Менеджер `Checkout` тимчасово резервує ("заморожує") товари на складі на час оплати та автоматично скасовує операцію (повертає товари) у разі виникнення помилок (`Exception`).
* **Персистентність даних**: Повне збереження та відновлення стану системи за допомогою бінарної серіалізації (`pickle`), читання зовнішніх лімітів із файлу конфігурації `config.json` та експорт фінансових звітів у формат `csv`.
* **Інтерактивний CLI-інтерфейс**: Зручне текстове меню на базі оператора `match/case` з високим рівнем стійкості до некоректного введення даних користувачем (`Fault Tolerance`).

## 🛠️ Вимоги до середовища

Для запуску проєкту та збирання пакунка необхідне наступне програмне забезпечення:
* Python версії 3.10 або новішої.
* Інструмент збирання `setuptools` або менеджер пакунків `pip`.

## 📦 Встановлення та розгортання проєкту

1. Клонуйте репозиторій із вихідним кодом на свій локальний комп'ютер:
   `git clone https://github.com/your-username/ecommerce-shopping-cart.git`
   `cd ecommerce-shopping-cart`

2. Перевірте наявність або створіть базовий конфігураційний файл `config.json` у кореневій директорії проєкту (поруч із `main.py`):
   `{ "store_name": "VNTU Tech-Shop International", "max_items_limit": 50 }`

3. Встановіть проєкт у локальне середовище в режимі розробки за допомогою конфігурації `pyproject.toml`:
   `pip install -e .`

## 🎮 Запуск програми

Для старту консольного інтерфейсу та початку взаємодії з системою виконайте команду:
`python main.py`
